Recommender System

Recommender systems are the systems that are designed to recommend things to the user based on many different factors.

Requirements
1. Opencv
2. tensorflow
3. sklearn
4. numpy.linalg
5. scipy
6. matplotlib
7. keras
8. pandas

Running
The scripts can be running in jupyter notebook or visual studio just by clicking run
It can also be run in terminal by typing python3 recommender

Dataset used
https://www.kaggle.com/datasets/olgabelitskaya/style-color-images/versions/7?resource=download
